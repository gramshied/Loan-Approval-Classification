# Loan Approval Prediction (Classification)

## Objective
To predict whether a loan application will be approved
based on applicant and financial attributes.

## Dataset
The dataset includes applicant income, loan amount,
credit history, and loan status.

## Tools Used
- Python
- Scikit-learn
- Pandas, NumPy

## Machine Learning Techniques
- Logistic Regression
- Regularization
- Train-test split
- Confusion matrix
- Precision, Recall, F1-score

## Results
- Credit history identified as a key predictor
- Model performance evaluated using classification metrics

## Conclusion
The project demonstrates understanding of supervised
classification problems and evaluation techniques.
